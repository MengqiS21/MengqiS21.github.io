# Image Folder Guide

Put all local images here and reference them with relative paths in HTML.

## Suggested structure
- `home/` : profile photo, homepage visuals
- `research/` : thumbnails, banners for research pages
- `publications/` : paper preview figures (optional)

## Path examples
- Profile image in homepage: `assets/images/home/profile.jpg`
- Research card thumbnail: `assets/images/research/human-ai-thumb.jpg`
- Research detail banner: `assets/images/research/human-ai-banner.jpg`

## Notes
- Keep filenames lowercase with hyphens, e.g. `human-ai-banner.jpg`.
- Prefer `.jpg` for photos, `.png` for transparent assets, `.webp` for size optimization.
