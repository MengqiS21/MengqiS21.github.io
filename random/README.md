# Random Pages Templates

This folder stores templates for the homepage `Random` links.

## Pages
- `photography.html`
- `design.html`
- `reading.html`
- `thoughts.html`

## How to edit
- Update text directly in each HTML file.
- Add local images under `../assets/images/` and reference by relative path.
  - Example: `../assets/images/home/my-photo.jpg`
- Shared style uses `../css/main.css`.

## Link source
Homepage links are configured in `../index.html` under `#random` section.
